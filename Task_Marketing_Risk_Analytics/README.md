# Marketing Risk Analytics Practical Task

This project analyzes marketing campaign risk using Python, Monte Carlo simulation, machine learning, LLM-based reputation scoring, and Power BI.

## Main Steps

1. Loaded and explored advertising campaign data.
2. Created marketing features such as ROI, CPR, Total Spend, and channel shares.
3. Ran 10,000 Monte Carlo simulations to estimate campaign risk.
4. Used Isolation Forest to detect unusual campaign records.
5. Scored social media posts for brand reputation risk.
6. Exported results to Excel.
7. Built an interactive Power BI dashboard.

## Key Findings

TV and Radio advertising showed stronger impact on Sales than Newspaper advertising.  
Monte Carlo simulation showed downside ROI risk.  
Anomaly detection found unusual campaign periods.  
Brand reputation scoring identified high-risk posts that require attention from marketing or PR teams.
